# Vaytracer

## Instal

```bash
git clone https://github.com/Saverio976/vaytracer.git vaytracer
cd vaytracer
make vautracer
```

## Usage

```
# show image in gui
./vaytracer --scene-file ./scenes/basic1.toml --gui
# output ppm image
./vaytracer --scene-file ./scenes/basic1.toml --output-file test.ppm
```
